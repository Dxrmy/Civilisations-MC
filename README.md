**Civilisations** is a profound overhaul of the Minecraft world-gen and entity systems. It breathes life into the world by introducing complex, ancient societies that build, defend, and evolve.

## Features
- 🏛️ **Procedural Ruins & Cities**: Exploration-focused world-gen that places architecturally significant structures.
- 💂 **Social AI**: Custom entity logic for Soldiers and Citizens that react to the player and their environment.
- 📜 **Historical Lore**: Discover artifacts and clues about the civilizations that came before.
- 🛠️ **Fabric Native**: Built for high performance on the latest Fabric toolchain.

## Tech Stack
- **Mod Loader**: Fabric API (1.21.11)
- **Library**: GeckoLib for advanced animations
- **Architecture**: Custom Entity & World-Gen Registry
